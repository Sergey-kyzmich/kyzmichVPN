Описание

Этот репозиторий содержит простой Telegram-бот для выдачи подписок на xray (VLESS/VMESS) и записи выданных подписок в PostgreSQL.

Файлы

- [bot.py](bot.py) — основной бот
- [schema.sql](schema.sql) — SQL-схема для создания таблиц
- [.env.example](.env.example) — пример переменных окружения
- [requirements.txt](requirements.txt) — зависимости

Установка

1. Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

2. Создайте базу данных PostgreSQL и примените схему:

```bash
psql "postgresql://user:password@localhost:5432/yourdb" -f schema.sql
```

или позвольте боту создать таблицы автоматически при первом запуске (бот выполнит `schema.sql`).

3. Скопируйте `.env.example` в `.env` или экспортируйте переменные окружения и заполните:

- `DATABASE_URL` — URL в формате `postgresql://user:pass@host:port/db`
- `BOT_TOKEN` — токен Telegram-бота
- `ADMIN_ID` — ваш Telegram user_id (для команды /addplan)
- `XRAY_HOST`, `XRAY_PORT`, `XRAY_UUID` — параметры вашего xray-сервера

Запуск

```bash
python bot.py
```

Команды бота

- `/start` — приветствие
- `/plans` — список тарифов
- Нажать кнопку тарифа — оформить подписку (сохраняется в БД)
- `/my` — показать ваши подписки
- `/addplan <name> <days> <price>` — добавить тариф (только админ)

Примечания

- Генерируемая ссылка — пример VLESS-ссылки. Для реальной интеграции необходимо, чтобы xray-сервер поддерживал динамическое создание/проверку учетных записей или у вас была серверная логика для управления пользователями на стороне xray.
- Этот пример не проводит оплату — ожидается, что вы интегрируете платёжный шлюз отдельно и после оплаты вызываете создание подписки.
